# Propuesta del proyecto web - Restaurante Xinyang

## Objetivo general

Desarrollar una página web funcional, organizada y visualmente atractiva para el restaurante chino Xinyang, utilizando HTML, Bootstrap 5, PHP, MySQL, XAMPP, Docker y arquitectura MVC.

## Descripción del restaurante

Xinyang es un restaurante chino que ofrece platos tradicionales como arroz chino tradicional, arroz chino con camarón, arroz valenciana, chow mein, espaguetti chino, chop suey, lumpias, combinados, platos especiales, adicionales y bebidas. La página web permite presentar su identidad visual, menú, horarios, métodos de pago y datos de ubicación.

## Tecnologías utilizadas

- HTML para la estructura del sitio.
- Bootstrap 5 para el diseño responsive.
- PHP para la lógica del servidor.
- MySQL para almacenar menú, reservas y mensajes.
- XAMPP para ejecución local.
- Docker para despliegue con contenedores.
- Arquitectura MVC para organizar el código.

## Arquitectura MVC

El proyecto está dividido en tres capas principales:

- **Modelo:** clases encargadas de la conexión con la base de datos y operaciones sobre menú, reservas y contacto.
- **Vista:** archivos que muestran la interfaz al usuario.
- **Controlador:** clases que reciben solicitudes, procesan datos y cargan las vistas.

## Funcionalidades principales

1. Página de inicio adaptada a la marca Xinyang.
2. Menú dinámico consultado desde MySQL.
3. Galería de imágenes suministradas en el documento del proyecto.
4. Formulario de reservas/pedidos con almacenamiento en base de datos.
5. Formulario de contacto con almacenamiento en base de datos.
6. Diseño adaptable a computador, tablet y celular.

## Conclusión

El proyecto cumple los requerimientos académicos porque implementa una página web funcional, usa PHP y MySQL, trabaja bajo MVC, permite ejecución con XAMPP o Docker e integra la identidad visual real del restaurante Xinyang.
